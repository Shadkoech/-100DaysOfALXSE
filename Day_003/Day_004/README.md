## Task

File:

	- split_list.c

Given a linked list, write a function that divides it into two halves. This function should modify the original list and then return a pointer to the second half of the list.

eg.

l = 1 -> 2 -> 3 -> 4

divided(l) = 3 -> 4
l = 1 -> 2
